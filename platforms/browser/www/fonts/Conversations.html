          Hi All,

          Just a reminder for next week and what you need to prepare.

          1. If you have any iOS device that you want to be able to test apps on then you need to send Steve the UDID and the type of device it is.
          2. Follow the instructions on the Apple Developer Account page in the modules to set up your Apple ID and join our team using the link in the email you were sent from Apple Developers.
          3. Create your CSR file and upload that to the apple developer account to create your Certificate.
          4. Next Wednesday I will create a new provisioning profile file for everyone to download and use.
          5. Update your copy of XCode
          6. Download the latest iOS simulator in XCode Preferences.
          7. Install the xcode command line tools so Cordova and XCode can communicate
          8. Install the npm ios-deploy tool.

          All this before next Wednesdays class.
          Cheers,
          Steve
